# Audio Demo Sample

## Purpose
This sample demonstrates how to use `com.gladfox.audiomanager` in a consumer project.

## Setup
1. Import this sample from Package Manager.
2. Run `Tools/Audio/Setup/Generate Production Assets`.
3. Open `Assets/Samples/Audio Manager/0.1.0/AudioDemo/Scenes/AudioDemoSampleScene.unity`.
4. Enter Play Mode and use on-screen controls.

## What is covered
- dynamic preload with progress overlay,
- UI and 3D SFX playback by id,
- music toggle and resume after sound re-enable,
- pause and snapshot transitions.

## Notes
- The setup tool generates required mixer/config/events/assets in `Assets/`.
- Package release channel for `0.1.x` is git tags (`upm/vX.Y.Z`).
