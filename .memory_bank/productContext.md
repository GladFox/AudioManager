# Product Context

## Назначение проекта
`AudioManager` задает единый слой управления аудио для команды разработки, чтобы геймплей, UI и контент-пайплайн работали через общий API и согласованный микс.

## Какие проблемы решает
- Устраняет разрозненные вызовы `AudioSource.Play*` по проекту.
- Убирает лишние `Instantiate/Destroy` аудио-источников в рантайме.
- Унифицирует громкости, mute, pause и snapshot-переходы.
- Дает саунд-дизайну настраиваемые события без изменения кода.

## Для кого предназначен
- Gameplay/UI программисты: простой API (`PlayUI/PlaySFX/PlayMusic`).
- Саунд-дизайнеры: настройка `SoundEvent` и микса через `AudioMixer`.
- Техлид/стюард: контроль архитектурной дисциплины, документации и качества.
